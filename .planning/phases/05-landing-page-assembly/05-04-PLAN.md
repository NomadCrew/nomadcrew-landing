<task type="auto">
  <name>Task 2: Verify JSON-LD schemas in page source</name>
  <files>None (verification only)</files>
  <action>
Fetch the deployed page and verify JSON-LD schemas are present in the HTML source.

```bash
# Fetch page source (use deployment URL from Task 1)
DEPLOY_URL="https://YOUR_HASH.nomadcrew-landing-page.pages.dev"
curl -s "$DEPLOY_URL" > /tmp/landing-page.html

# Check for JSON-LD scripts
grep -o 'application/ld+json' /tmp/landing-page.html | wc -l
# Should be 2 (Organization + WebSite)

# Extract and validate Organization schema
grep -oP '(?<=<script type="application/ld\+json">).*?(?=</script>)' /tmp/landing-page.html | head -1

# Extract and validate WebSite schema
grep -oP '(?<=<script type="application/ld\+json">).*?(?=</script>)' /tmp/landing-page.html | tail -1

# Validate OG tags
grep 'og:image' /tmp/landing-page.html
grep 'og:title' /tmp/landing-page.html
grep 'og:description' /tmp/landing-page.html
```

Verify:
1. Two JSON-LD script blocks present
2. Organization schema has @type, name, url, logo
3. WebSite schema has @type, name, url
4. OG image URL is absolute (https://nomadcrew.uk/og-image.jpg)
  </action>
  <verify>
```bash
# Count JSON-LD blocks (should be 2)
curl -s "$DEPLOY_URL" | grep -c 'application/ld+json'

# Verify Organization type
curl -s "$DEPLOY_URL" | grep -o '"@type":"Organization"'

# Verify WebSite type
curl -s "$DEPLOY_URL" | grep -o '"@type":"WebSite"'

# Verify OG image
curl -s "$DEPLOY_URL" | grep -o 'og:image.*content="[^"]*"'
```
