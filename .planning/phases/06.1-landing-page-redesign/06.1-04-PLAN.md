3. Create SocialProof component:

   a. **State (if using client-side fetch):**
      ```typescript

      const [loading, setLoading] = useState(true);
      ```

   b. **Fetch effect (optional, can be static initially):**
      ```typescript

          .then(data => setCount(data.count))
          .catch(() => setCount(null))
          .finally(() => setLoading(false));
      }, []);
      ```

      - If loading: show subtle skeleton or placeholder
      - If failed/null: show generic "Join travelers worldwide" or hide count entirely

      - Centered section with warm background (design token)
      - Large, confident typography

      - Optional: trust badges (e.g., "Early access", "No credit card required")
      - Consider using Users icon to reinforce community feel

   e. **Example Structure:**
      ```tsx
      <section className="py-12 px-4 bg-[var(--color-bg-secondary)]">
        <div className="container mx-auto text-center">
          <motion.div
            initial={{ opacity: 0, y: 20 }}
            whileInView={{ opacity: 1, y: 0 }}
            viewport={{ once: true }}
          >
            <div className="flex items-center justify-center gap-2 mb-4">
              <Users className="w-6 h-6 text-orange-500" />
              <span className="text-lg font-semibold text-gray-700">
                {count ? `${count.toLocaleString()}+ travelers` : 'Travelers worldwide'}
              </span>
            </div>
            <p className="text-gray-600 max-w-md mx-auto">
              Join the community of adventurers planning their next group trip with NomadCrew
            </p>
          </motion.div>
        </div>
      </section>
      ```

   f. **Trust Indicators (optional additions):**

      - "Join the beta"
